# radio-dev
 
Local Radio service with AI generated podcasts each hour !

---

## Requirements

| Tool | Role | Install |
|------|------|---------|
| Docker | Run the service | [docs.docker.com](https://docs.docker.com/engine/install/) |
| Docker Compose | same as above | [docs.docker.com](https://docs.docker.com/compose/install/) |
| [podcast-generator](https://github.com/AdrienPiechocki/podcast-generator.git) | podcast generation | `git clone https://github.com/AdrienPiechocki/podcast-generator.git /path/to/radio-dev/` |

## Setup

```bash
docker compose up -d
```

## Usage

visit this URL : http://localhost:8080/
