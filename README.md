# radio-dev
 
Local Radio service with AI generated podcasts and news in French !

---

## Requirements

| Tool | Role |
|------|------|
| Docker | Run the service |
| Docker Compose | same as above |
| Git | clone needed repos |

## Setup

```bash
docker compose up -d
```

## Usage

visit this URL : http://localhost:8080/

## Troubleshoting

Q: No music ??

A: You must put your own .mp3 files in the music subfolder (create it first)

Q: generator not working

A: remove the .venv inside the generator (it MUST be generated by Docker)
